# Project Management Tool

A lightweight, intuitive web application designed to streamline task allocation and management across multiple active projects. 

## 🚀 Live Demo
Check out the live project here: [Project Management Tool](https://bhuvaneswari123457.github.io/project-management-tool/)

## ✨ Features
* **Project Selection:** Easily categorize and assign tasks to specific ongoing projects (e.g., TO DO LIST, BLOG WEBSITE, PORTFOLIO WEBSITE).
* **Task Creation:** Clear, straightforward input fields for outlining specific task descriptions.
* **Team Delegation:** Directly assign responsibilities to specific team members to keep track of who is doing what.
* **Clean UI:** A simple, distraction-free interface optimized for quick task entries.

## 🛠️ Technologies Used
* HTML5
* CSS3
* JavaScript
